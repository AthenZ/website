Contact / Support

GitHub

Athenz is completely open source and can be found here on Github. We welcome any issues and pull requests!

Slack

We use Slack for discussion and support. For any Athenz questions, join the `#general` channel on the Athenz Slack team. For everything else, join the `#random` channel.

You may also use the following discussion groups:

  • Athenz-Dev for development discussions
  • Athenz-Users for users questions

Stack Overflow

We monitor Stack Overflow for any posts tagged with screwdriver-cd. If there aren't any existing questions that help with your problem, feel free to ask a new one!
